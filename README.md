## Getting Started

Inicializar la aplicacion:
1. Debemos iniciar el servidor haciendo
--> Las librerias en la parte del server son
   + cv2
   + pandas
   + mediapipe
   + flask
   + flask_cors
   + os 
   ```sh
   cd .\server\
   python .\main.py
   ```
3. Debemos iniciar el cliente haciendo
   ```sh
   cd .\client\
   npm run dev
   ```
